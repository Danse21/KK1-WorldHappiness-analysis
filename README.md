# Kort: vad datasetet är, varifrån det kommer, hur notebooken körs

**What is the dataset?**
The dataset is the Year 2019 version of the annual World Happiness Report published by FN's Sustainable Development Solutions Network (SDSN).

The data itself comes from Gallup World Poll, an opinion collection organisation that randomly selected or sample citizens of different countries and ask them asnwer certain question of interest.

**Where did the dataset come from?**
The dataset was obtained from Kaggle : https://www.kaggle.com/datasets/unsdsn/world-happiness?select=2019.csv

**How to run the notebook**

- Open the `notebook.ipynb`file in a VS code or Jupyter notebook code editor.
- if you choose VS code,
  - prepare the editor for python work.
  - Type "> Python: Create Environment....." and press Enter to run
  - Type "> Python: select interpreter" and select your python version - most recommended 3.14.4
- Click on "Run All" to run all the codes
